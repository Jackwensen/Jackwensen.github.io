week 8 Web ( HTML CSS JS )

  • internet 互联网 是互相通信的计算机网络，它提供了发送0和1的基础设施。在此基础上，我们可以构建发送和接收数据的应用程序.
  • Routers 路由器是具有 cpu 和内存的专用计算机，其目的是在互联网上的其他设备之间通过电缆或无线技术传递数据.
  • Protocols 协议是一系列标准的约定，比如物理握手，这是全世界都同意的，计算机可以与之通信的约定。例如，一台计算机必须使用某些零和一或消息的模式来告诉路由器它想把数据发送到哪里
  • TCP/IP是在两台计算机之间发送数据的两个协议。在现实世界中，我们可能会在信封上写上地址，以便寄信给某人，同时附上我们自己的地址，作为回信。一个信封的数字版本，或者一个带有发送地址和发送地址的消息，被称为packet 包.
  • IP是因特网协议的缩写，是现代计算机软件支持的一种协议，其中包括计算机互相寻址的标准方式IP addresses是连接到互联网的计算机的唯一地址，这样，从一台计算机发送到另一台计算机的数据包将沿路由器传递，直到到达目的地.
    • 路由器在其内存中有一个表，将 IP 地址映射到每个连接到其他路由器的电缆，这样它们就知道将数据包转发到哪里。路由器也有协议来进行通信，并找出这些路径
  • TCP传输控制协议，通过使用port number端口号（一个整数添加到 IP 地址后面，例如，HTTP、 HTTPS、电子邮件、甚至 Zoom 都有自己的端口号，这些程序可以使用它们在网络上进行通信），传输控制协议允许一个服务器在同一个 IP 地址上来提供多个服务
    • TCP 还提供了一种机制，当数据包以某种方式丢失或未接收时，可以重新发送数据包。事实证明，在互联网上，一个数据包有多条发送路径，因为有许多路由器是相互连接的。因此，一个网络浏览器向一只猫发出请求时，可能会看到其数据包通过路由器的一条路径发送，而响应服务器可能会看到其响应数据包通过另一条路径发送
    • 大量的数据，例如图片，将被分割成更小的块，这样所有的数据包都是相似的大小。通过这种方式，互联网上的路由器可以更公平、更容易地发送每个人的数据包。Net neutrality 网络中立性是指这些公共路由器平等地对待数据包，而不是允许来自某些公司或某些类型的数据包被优先化。
    • 当一个响应有多个数据包时，TCP 还将指定每个数据包都标记为“1 of 2”或“2 of 2”，以便根据需要将它们组合或重发
  • Web是一个运行在互联网之上的应用程序，允许我们获得网页。其他应用如 Zoom 提供视频会议，电子邮件也是一个应用
  • HTTP超文本传输协议控制着网络浏览器和网络服务器在 TCP/IP 数据包内的通信方式。
    • HTTP请求头尾，HTTP版本，状态码
  • HTML&CSS：Dom tree，标签，class，id
  • JavaScript ：基本用法，操作dom
